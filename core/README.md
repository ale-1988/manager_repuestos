Módulo Core

Este módulo contiene componentes transversales al sistema que no pertenecen a un dominio funcional específico.
Responsabilidades

    Alojar modelos base o utilitarios comunes

    Centralizar lógica compartida entre aplicaciones

    Evitar duplicación de código transversal

Consideraciones

    No debe contener lógica de negocio específica

    Su contenido suele crecer con el proyecto
