Este es el código fuente del desarrollo MANAGER DE REPUESTOS, trabajo de pregrado de Alejandro Gaut, año 2025.
Mas abajo se listan los paquetes utilizados en el desarrollo, que se instalan en sus versiones actualizadas.
Las credenciales estan almacenada de manera segura en un archivo .env que _no_ está disponible en el repositorio.

Paquetes a instalar:

python-django 
python-django-environ 
python-django-extensions
python-django-allauth 
python-django-crispy-forms 
python-crispy-bootstrap5 
python-reportlab
mariadb
mariadb-libs
mariadb-clients
apache
php-apache
phpmyadmin

Scripts auxiliares:

full_reset.sh: reinicio completo de entorno
limpiar_migraciones.sh: limpieza de migraciones
limpiar_pyc.sh: limpieza de artefactos Python
