Este es el código fuente del desarrollo MANAGER DE REPUESTOS, trabajo de pregrado de Alejandro Gaut, año 2025.
Para correrlo, debera contar con una instalación de django standard, y un servidor de BD MariaDB.
Si el servidor esta corriendo en local, el proyecto deberia correr sin tocar ninguna configuración, si no es asi, 
deberá editar la configuracion de las bases de datos en /manager_repuestos/settings.py indicando alli la 
dirección IP del host, y la password de la cuenta root.
